Mindset

How I Think About Security

Security is not just detection — it's understanding why an attack worked, what the attacker was after, and how to make the next attempt harder.

I focus on patterns across alerts, not isolated incidents. One phishing kit showing up across five campaigns tells a more useful story than five separate case closes.

Manual analysis doesn't scale. I'm interested in automating the repetitive parts — enrichment, correlation, reporting — so analysts can spend time on what actually requires judgment.

The best defense is built by people who've studied the offense. I read attacker infrastructure the same way an architect reads blueprints.
