PhishScan — Phishing Detection CLI

Built after spending hours manually parsing phishing emails in SOC workflows — this automates the repetitive parts.

A Python CLI tool that analyzes email headers, URLs, and domain patterns to score phishing likelihood. Extracts IOCs and generates incident-ready reports.

ThreatBoard — IOC Dashboard

Born from the need to see attacker infrastructure as a connected graph, not a flat list of indicators.

LogLens — SIEM Log Correlator

Written to cut down the time spent eyeballing raw log exports when looking for lateral movement patterns.
