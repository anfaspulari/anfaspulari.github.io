PhishScan — Phishing Email Analyzer

Python CLI for SOC analysts. Parses .eml files, extracts IOCs, applies 8 detection rules, and outputs a scored incident report. Below is a live simulation against a realistic phishing sample.

INPUT EMAIL sample_emails/phishing_sample.eml 
From:     support@secure-update-portal.com
Reply-To: attacker@fake-domain.ru
To:       victim@company.com
Subject:  Urgent: Action Required - Verify Your Account Immediately
Date:     Mon, 14 Jan 2024 09:23:11 +0000
Received-SPF: fail (domain of support@secure-update-portal.com does not
    designate 185.220.101.47 as permitted sender)

We have detected unusual login activity on your account...
Verify here: https://secure-login-update.com/verify?token=AbC123XyZ9f2
Support:     https://bit.ly/3xHelpDesk
bash — phishscan
Note: This is a browser simulation of the CLI output. Run the real tool locally: python phishscan.py sample_emails/phishing_sample.eml --verbose — install via github.com/anfaspulari/phishscan-cli
